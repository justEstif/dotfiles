Look deeper into git history to see how project interests changed over time

## Summary of Changes\n\nAnalyzed deeper history. The project started as a more typical dotfiles repository configuring various developer tools (Zellij, Zed, Neovim, Ghostty, OpenCode, Claude Code). Over time, the focus explicitly shifted away from other AI tools (Claude Code, OpenCode) and terminal multiplexers (Zellij) toward building an integrated, agent-first development environment centered almost entirely around the `pi` agent harness, `beans` issue tracker, and custom tools like `qry` and `defuddle`.
