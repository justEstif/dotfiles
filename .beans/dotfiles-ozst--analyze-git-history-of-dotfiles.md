## Summary of Changes

Analyzed the git history. Recent development has been highly focused on AI coding agents and extending the `pi` harness. Key themes:
- **Observational Memory (OM)**: Building an extension for `pi` inspired by Mastra to provide resource scope and observer/reflector mechanics.
- **Agent Strictness/ESLint**: Creating and enforcing strict ESLint rules (React, Next.js, Total TypeScript, Ousterhout software design principles) specifically designed for AI agents. This includes a dedicated npm package (`eslint-config-agent-strict`).
- **Tools/Plugins**: Improving plugins for `beans` and `qry`, adding `subagent` support to `pi`, and introducing new tools like `btw`.
